# Site Rodrigo Morelli — Protótipo

Protótipo navegável do site de Rodrigo Morelli: músico, compositor, instrumentista e referência nacional em banjo.

**Ver online:** https://SEU-USUARIO.github.io/site-rodrigomorelli

## O que é isto

Protótipo de avaliação de direção visual e arquitetura de conteúdo. Os textos são finais e podem ir para produção. Preços, vídeos e alguns blocos são placeholders.

## Estrutura

```
index.html      página única, HTML+CSS, sem dependências
assets/         imagens (logo, fotos, banjo Irineu, patrocinadores)
.nojekyll       impede o GitHub Pages de processar como Jekyll
```

Sem build, sem framework, sem npm. Abrir `index.html` no navegador funciona.

## Seções

Hero · Números · Curso de banjo · Loja de instrumentos · Sobre · Regulagem · Encontro de Banjeiros · Música · Patrocinadores · Rodapé

## Identidade visual

Preto e branco puro, extraída do logotipo existente. Tipografia Archivo em caixa alta. A cor entra apenas pelas fotografias.

| Token | Valor |
|---|---|
| Fundo | `#000000` |
| Painéis | `#0B0B0B` / `#111111` |
| Linhas | `#242424` |
| Texto | `#FFFFFF` / `#B8B8B8` / `#8C8C8C` |
